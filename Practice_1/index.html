Q1: What is Docker?

Docker is a Centralized platform for deploy and run the application. We can directly access the project without install dependences everytime and with just one command we can access the whole project or shared files. It is an OS level virtualization syatem which is light-weight.

Q2: What is Container?

The container is the heart of the docker which will contain the image of the code which give us the simple layout of the code, also container include packages to run the code, system tool etc.

Q3: What are Images?

Images are the basic layout of any code and based on the data of the image the browser will show the output.

Q4: Whats id Nginx?

The Nginx is the open source server which help to run the code on browser and provide the stability to the code.
